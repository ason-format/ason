MCP Server

Model Context Protocol Integration

GitHub

Official MCP server that enables ASON compression/decompression for AI assistants like Claude Desktop, Cline, Continue, and other MCP-compatible clients.

Available Tools

•
compress_json - Compress JSON to ASON
•
decompress_ason - Decompress to JSON
•
get_compression_stats - Analyze metrics
•
configure_compressor - Update settings
npx @ason-format/mcp-server

VS Code Extension

Visual Studio Code Integration

Marketplace

Compress and decompress JSON directly in VS Code with keyboard shortcuts, context menus, and automatic MCP configuration for GitHub Copilot.

Features

  • • Compress Selection Cmd+Alt+C
  • • Decompress Selection Cmd+Alt+D
  • • Show Statistics Cmd+Alt+S
  • • Status bar indicator with real-time MCP status
  • • Automatic GitHub Copilot MCP configuration
  • • Right-click context menu integration
ext install ason-format.vscode-ason

Zed Extension

High-Performance Zed Editor

GitHub

Native Zed extension providing ASON compression with slash commands and seamless integration with Zed's AI assistant.

Features

  • • /ason-compress - Compress selected JSON
  • • /ason-decompress - Decompress ASON to JSON
  • • /ason-stats - View compression statistics
  • • Built with Rust for maximum performance
  • • Direct integration with Zed AI assistant
zed: extensions -> install "ASON"
